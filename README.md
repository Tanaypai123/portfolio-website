# Tanay Sharma — Personal Portfolio

A world-class personal portfolio website for Tanay Sharma, Full Stack Developer.

## Tech Stack

- **Next.js 15** — React framework
- **TypeScript** — Type safety
- **Tailwind CSS** — Utility-first styling
- **Framer Motion** — Smooth animations
- **Lucide React** — Icons

## Design

- Dark theme (`#0A0A0F` background)
- Purple accent (`#7C3AED`) + Indigo secondary
- Glassmorphism cards with `backdrop-filter: blur`
- Inter font via Google Fonts
- Fully responsive (mobile, tablet, desktop)

## Sections

1. **Hero** — Name, tagline, status, CTA buttons, social links
2. **About** — Bio, education, currently building/learning/goal
3. **Featured Products** — FlowStock (primary), HireBlind (secondary)
4. **Experience** — Timeline with INOVIZE IT SOLUTIONS & Kistenchosoftware
5. **Skills** — Categorized tech stack
6. **Achievements** — Hackathon cards
7. **GitHub** — Profile + repository showcase
8. **Contact** — Email, LinkedIn, GitHub, Phone

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

This site is ready to deploy on:
- **Vercel** (recommended) — `vercel deploy`
- **Cloudflare Pages** — connect GitHub repo
- **Netlify** — drag & drop or connect repo
